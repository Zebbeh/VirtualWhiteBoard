Api Documentation

Endpoints

  • USERS

    • GET User Information: GET /
    • Create User: POST /
    • User Login: POST /login
    • Update User: PATCH /ID
    • Delete User: DELETE /ID

  • POSTS

    • Post a New Post: POST /
    • Patch a Post: PATCH /ID
    • Delete a Post: DELETE /ID

  • BOARDS
    • GET boards the user has access to: GET /user/userId
    • GET board with id: GET /ID
    • Create a new board: POST /
    • NEED TO IMPLEMENT PATCH, TO GET THE NOTES THAT ARE POSTED

How to use the API

First you need to create an account via login page.
Then you can login, after login you will get your token that you need for rest of the API.
